# Portfolio

## Project info

A minimal and elegant portfolio website.

## How to run locally

The only requirement is having Node.js & npm installed.

Follow these steps:

`sh
# Step 1: Install the necessary dependencies.
npm i

# Step 2: Start the development server.
npm run dev
`

## Technologies Used

- Vite
- React
- Tailwind CSS
- Framer Motion
- Lucide React
- Radix UI
